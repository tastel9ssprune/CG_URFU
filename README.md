# Компьютерная графика - Лабораторные работы

Репозиторий с лабораторными работами по курсу компьютерной графики.

## Структура

- `lab1/` - Рисование простых фигур (Windows Forms)
- `lab2/` - Рисование фигур в OpenGL (OpenTK)
- `lab3/` - Построение графиков функций (OxyPlot)
- `lab4/` - Фракталы (треугольник Серпинского, множество Мандельброта)
- `lab5/` - Работа с изображениями (Bitmap)
- `lab6/` - 3D фигуры с вращением (OpenGL)
- `lab7/` - 3D дерево с управлением камерой (OpenGL)
- `lab8/` - Raycasting 3D сцены

## Запуск

Для каждой лабораторной работы смотрите README в соответствующей папке.
